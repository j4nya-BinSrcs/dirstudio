# 🚀 DirStudio

> A modern directory intelligence platform for analyzing, organizing, and optimizing your filesystem.

DirStudio is a full-stack application that provides deep insights into your file system — helping you analyze storage, detect duplicates, and leverage AI to organize your directories intelligently.

---

## 📌 About

Managing files at scale becomes messy, inefficient, and error-prone. DirStudio solves this by combining:

- 📊 **Data visualization** for filesystem insights  
- 🧹 **Duplicate detection** to reclaim storage  
- 🤖 **AI-powered organization** for smarter structure  

It is designed as both:
- a **practical productivity tool**, and  
- a **systems-focused engineering project**

---

## ✨ Key Capabilities

- Directory analytics (files, size, structure)
- Duplicate file detection and cleanup
- File type distribution and insights
- AI-assisted file organization (Mistral)
- Interactive web dashboard
